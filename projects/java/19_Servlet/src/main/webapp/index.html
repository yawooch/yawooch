javax.servlet.http.HttpServlet 클래스를 상속 받아야한다.

3. HTTP 요청 처리

1) GET 방식은 데이터를 URL에 포함시켜 요청하는 것으로 보안 유지가 불가능하다
2) POST 방식은 데이터를 URL에 노출하지 않고 요청하는 것으로 보안 유지가 가능하다
