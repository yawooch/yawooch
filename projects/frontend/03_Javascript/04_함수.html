1. 함수 선언

1) 선언적 함수

function 키워드에 함수명을 작성하여 선언하는 방법이다.

2) 익명 함수

함수명을 작성하지 않고 선언하는 방법이다.

3) 화살표 함수

ES6에 도입된 함수 선언 방법으로 function 키워드 대신 =>를 사용해서
좀 더 간략한 방법으로 함수를 선언할수 있다

2. 함수 호출

1) 매개 변수

함수 실행에 필요한 값은 매개 변수를 통해 함수에 전달할 수 있다.

2) arguments 객체

초과되는 매개값은 arguments 객체를 통해 접근이 가능하다.
