객체

1. 객체 생성

자바스크립트는 다양한 방법으로 객체를 생성할 수 있다.

2. 프로퍼티

프로퍼티는 키(key)와 값(value)으로 구성된다.

3. 메소드

프로퍼티의 값이 함수인 경우 메소드라 한다.

4. 객체에 속성 추가 / 삭제

메소드는 객체 생성 후 동적으로 추가할 수 있다

5. 객체 배열

6. 생성자 함수

new 연산자와 함께 호출되어 객체를 생성하는 함수를 생성자 함수라 한다.

7. 캡슐화

지역변수와 클로저를 활용하여 캡슐화를 구현할 수 있다.

8. Date 객체

날짜와 시간 정보를 저장하는 객체이다.
