CSS선택자1

선택자란 ?
특정 HTML 요소를 선택할 떄 사용하는 기능으로, 요소를 선택하여 원하는 스타일을 적용할수 있다.

1. 전체 선택자

HTML 문서 내부에 모든 태그를 선택할 때 사용한다 

            * {
                스타일 속성 : 값;
                스타일 속성 : 값;
                ...
            }

3. 아이디 선택자

HTML 문서 내부에 해당하는 아이디 속성값을 가진 태그만 선택할때 사용한다. 

            #아이디 {
                스타일 속성: 값;
                스타일 속성: 값;
            }

4. 클래스 선택자

HTML 문서 내부에 해당하는 클래스 속성값을 가진 태그들을 선택할때 사용한다. 

            .클래스 {
                스타일 속성: 값;
                스타일 속성: 값;
            }
